# laravel-nuxt-bs

Laravel 7 + NuxtJS + BootstrapVue with ESLint with VSCode support.

## Installation

Clone and setup: 

```bash
$ git clone https://github.com/ordigital/laravel-nuxt-bs.git your-project
$ cd your-project
$ npm install
$ npm audit fix
$ php artisan key:generate
```
Run developement server:
```bash
$ npm run dev
```
Build for production:
```bash
$ npm run build
```
