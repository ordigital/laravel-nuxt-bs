# Nuxt Laravel Starter

Laravel 7 + NuxtJS + BootstrapVue

## Scripts

```bash
# *** Global ***
# development build with nuxt dev server with hot reloading
npm run dev
# production build
npm run build
# perform tests
npm run test
# find linting errors
npm run lint
# autofix linting errors
npm run lint:fix
# check for conflicting lint rules
npm run lint:check
```
